# mark-calculator
This is a small website to be able to check what you need in an exam to be able to get an A overall

## How to Access on Mobile

1. Ensure your computer and phone are on the same Wi-Fi network.
2. Run a local server on your computer:
   ```
   python -m http.server 8000
   ```
3. Find your computer's local IP address (e.g., using `ipconfig` on Windows).
4. On your phone, open a web browser and navigate to `http://<your-ip>:8000` (replace `<your-ip>` with your actual IP address, like `192.168.18.92`).
5. The site should load and be fully functional on your mobile device.
